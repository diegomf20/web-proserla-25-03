# web-proserla

## Instalar Librerias
```
npm install
```

### Correr en entorno de desarrollo
```
npm run serve
```

### (Build) Compilar vistas para production
```
npm run build
```

### Despliegue en el servidor
<p>Despues del "build", Copiar contenido de la carpeta disk a la raiz de la carpeta publica en el servidor.</p>
